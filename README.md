# Первый проект по курсу "Операционные системы"

## Задание

1. Создать вектор из **1 000 000 элементов** и наполнить его случайными числами.  
2. Посчитать **сумму этого вектора на одном потоке** — просто в цикле.  
3. Замерить **время работы программы**.  
4. Посчитать сумму вектора **по частям**, разделив его на 4 части (по 250 000 элементов) и запустив **4 потока**, каждый из которых суммирует свой участок.  
5. Снова замерить время и **сравнить** результаты с однопоточным выполнением.

## Реализация

Программа реализует **два подхода** для сравнения производительности.

### 1. `func_with_one_general_vec()`
- Суммирование в **одном потоке**.  
- Затем суммирование в **4 потоках**, при этом все потоки работают с **одним и тем же вектором**, но с разными диапазонами индексов.  
- Используются **лямбда-выражения** с захватом внешних переменных, **без внешних функций**.

### 2. `func_with_four_vec_thread()`
- Суммирование в **одном потоке**.  
- Затем суммирование в **4 потоках**, но при этом для каждого потока создаётся **отдельный под-вектор** (т.е. используется дополнительная память).  
- Применяются **внешние функции** с передачей переменных **по ссылке**, **без лямбда-выражений**.

Обе функции выполняют **замер времени** и **сравнение скорости** выполнения при одинаковых данных.
